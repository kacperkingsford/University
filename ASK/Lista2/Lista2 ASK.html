x,y>0
Bit znaku x i y to 0

2(2^31 - 1) >= x+y > 2^(31)
binarnie:
0|11…1 >= x+y > |011…

Zadanie8

x = (x >> 31) - (-x >> 31);

Uzasadnienie:
x >> 31 : 0 dla x = 0, 0 dla x > 0, -1 dla x < 0 (111…111) = -1
-x >> 31: 0 dla x = 0, -1 dla x > 0 (111…111) = -1 , 0 dla x < 0

Łatwo zauważyć, że nie zadziała dla x = INT_MIN.

x = (x >> 31) | (-x >> 31 & 1);

Uzasadnienie:

  • dla x = INT_MIN

    • x >> 31 = -1
    • -x >> 31 <=> (-INT_MIN) = INT_MIN << 31 = -1
    • -x >> 31 & 1 = 1

  • x >> 31 :

    • 0 dla x = 0,
    • 0 dla x > 0,
    • -1 dla x < 0 (111…111) = -1

  • -x >> 31 & 1:

    • 0 dla x = 0,
    • 1 dla x > 0 (111…111) & 1 = 1 ,
    • 0 dla x < 0 (0…) >> 31 & 1 = 0 & 1 = 0

W takim razie mamy:

  • 0 | 0 = 0 gdy x = 0
  • 0 | 1 = 1 gdy x > 0
  • -1 | 0 = -1 gdy x < 0
  • -1 | 1 = -1 dla x = INT_MIN
